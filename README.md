Trabalho Python

# Comandos importantes
```shell

# Cria o venv
py -3 -m venv .venv

# habilita do venv
.venv\Scripts\activate

# Instala flask-login
pip install flask-login

# Instala flask-sqlalchemy
pip install -U Flask-SQLAlchemy

# Instala flask-wtf
pip install -U Flask-WTF

# Instala o flask
pip install flask

# Roda a aplicação
flask --app main run

```
